Detailed Description

Event queue module for decoupling interrupt handlers from UI processing.

Author
Dennis Rathgeb
Date
Apr 26, 2024

This module implements a linked-list FIFO queue for events generated by button presses and encoder rotation. Events are enqueued in ISR context and dequeued in the main loop for UI processing.

Note
Uses dynamic memory allocation (malloc/free) which may not be ideal for real-time embedded systems. Consider using a static memory pool.
