Detailed Description

Implementation of menu-driven user interface.

Author
Dennis Rathgeb
Date
Apr 20, 2024

Implements a state machine-based menu system for kiln control. Handles navigation, program creation, and settings adjustment via LCD display and rotary encoder input.
