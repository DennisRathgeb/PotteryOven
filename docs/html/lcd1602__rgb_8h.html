Detailed Description

LCD1602 RGB display driver with I2C interface.

Author
Dennis Rathgeb
Date
Apr 6, 2024

Driver for the Grove LCD1602 RGB backlight display. The display has two I2C addresses:

  • LCD controller at 0x7C
  • RGB backlight controller at 0xC0

Supports 4-bit mode operation with full control over display, cursor, scrolling, and RGB backlight color.

Performs the complete LCD initialization sequence:

  1. Wait for power stabilization (50ms)
  2. Send function set command 3 times (per HD44780 spec)
  3. Configure display control, clear display, set entry mode
  4. Initialize RGB backlight to white
