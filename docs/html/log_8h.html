Detailed Description

Logging module with configurable log levels and UART output.

Author
Dennis Rathgeb
Date
Nov 15, 2023

This module implements a logging feature with multiple logging levels: LOG_DEBUG, LOG_INFO, LOG_WARNING, LOG_ERROR

Output is sent to UART (handle passed to initLog). Printf is rerouted to UART if REROUTE_PRINTF is defined.
