Detailed Description

User interface module for menu-driven LCD control.

Author
Dennis Rathgeb
Date
Apr 20, 2024

Implements a state machine-based menu system for the kiln controller. Uses a 16x2 LCD display and rotary encoder with buttons for navigation.
